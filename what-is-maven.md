# Maven是什么

Maven提供了大量有用的项目信息，这些信息部分来自您的POM，部分来自您项目的来源。 例如，Maven可以提供：

更改直接从源代码管理创建的日志文档
交叉引用的来源
邮件列表
依赖列表
单元测试报告包括报道
随着Maven的改进，所提供的信息集将得到改善，所有这些对Maven的用户都是透明的。

Maven本质上是一个项目管理和理解工具，因此提供了一种帮助管理的方法：

<ul>
<li>Builds</li>
<li>Documentation</li>
<li>Reporting</li>
<li>Dependencies</li>
<li>SCMs</li>
<li>Releases</li>
<li>Distribution</li></ul>
